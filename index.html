Un pequeño resumen
Mi formación en diferentes campos de la programación me hace esta perfectamente cualificado para prácticamente cualquier proyecto. Además, siempre estoy dispuesto a aprender ya que valoro adquirir conocimiento y experiencia en cualquier lenguaje de programación o framework.
Los lenguajes de programacion que conozco Ahora algo de información mas técnica sobre los lenguajes de programación que conozco.
Java
Tengo cuatro años de experiencia en programación con Java, aunque aún no he trabajado profesionalmente. Durante mis prácticas en Atos, participé en un proyecto para el seguimiento y evaluación de alumnos en prácticas, utilizando Spring Boot para la API y Angular para el front-end. Tengo experiencia con frameworks como Spring Boot, JPA, Hibernate y en la creación de APIs y llamadas directas a bases de datos MySQL. Además, he trabajado con JUnit para pruebas y depuración de código. Actualmente, estoy desarrollando un proyecto personal junto a un compañero, donde utilizamos GitHub para colaborar y trabajar simultáneamente en diferentes aspectos del código.
JavaScript
Tengo cuatro años de experiencia en programación con JavaScript, aunque tampoco he trabajado profesionalmente de ello. Me he especializado en desarrollo web, creando páginas interactivas utilizando JavaScript junto con jQuery, jQuery UI y AJAX. Principalmente, he trabajado con el framework Angular para desarrollar aplicaciones web robustas. Además, tengo experiencia con tecnologías relacionadas como HTML, CSS y Sass, lo que me permite crear interfaces de usuario atractivas y funcionales. He utilizado Jasmine para la depuración y pruebas de código JavaScript, asegurando la calidad y el rendimiento de mis aplicaciones. Actualmente estoy desarrollando un proyecto personal para poner a prueba mis conocimientos. Este portfolio es un ejemplo de ello.
Php
Tengo cuatro años de experiencia en programación con PHP por mi cuenta, enfocándome en el desarrollo web tanto en el backend como en el frontend. He trabajado en la creación de APIs propias y en el consumo de APIs externas, lo que me ha permitido desarrollar aplicaciones web robustas y funcionales. Principalmente, he utilizado el framework Laravel para estructurar y optimizar mis proyectos. Además, tengo experiencia trabajando con bases de datos MySQL, lo que me permite gestionar y manipular datos de manera eficiente. La primera vez que trabajé en equipo fue con PHP, utilizando GitHub para colaborar y desarrollar una aplicación funcional, lo que me ayudó a familiarizarme con el trabajo en equipo y el control de versiones.
Python
Tengo tres años de experiencia en programación con Python, enfocándome principalmente en inteligencia artificial y análisis de datos. He trabajado en desarrollo web utilizando el framework Django, lo que me ha permitido crear aplicaciones web robustas y escalables. Además, tengo experiencia con bibliotecas como NumPy, Pandas, Matplotlib y Seaborn para el análisis y visualización de datos. En el ámbito de la inteligencia artificial, he utilizado Keras, TensorFlow y Scikit-Learn para desarrollar y entrenar modelos de aprendizaje automático. También he trabajado con bases de datos MariaDB, lo que me permite gestionar y manipular datos de manera eficiente. Mi experiencia en Python me ha permitido abordar una amplia variedad de proyectos y desafíos en el campo de la programación.
TypeScript
Tengo experiencia en programación con TypeScript, principalmente en el desarrollo de aplicaciones web utilizando Angular. TypeScript me ha permitido escribir código aprovechando sus características de tipado estático y herramientas de desarrollo avanzadas.
SQL
He trabajado con SQL en diversos proyectos y lenguajes de programación, incluyendo Java, JavaScript y PHP. Tengo experiencia en la gestión y manipulación de bases de datos MySQL y MariaDB, lo que me permite diseñar y optimizar consultas eficientes para diferentes aplicaciones.

También he trabajado en la creación de diseños responsivos y adaptativos, asegurando que las aplicaciones se vean bien en diferentes dispositivos y tamaños de pantalla. En todos los proyectos grupales en los que he participado, ha sido necesario trabajar con CSS para lograr una presentación visual coherente y profesional.
